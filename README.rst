``CTF``: Composite Template Fitting Software
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
This repository provides an SED fitting tool, primarily focused on the IR part of the spectrum.


Requirements: 
~~~~~~~~~~~~~
    .. code:: 
    

       numpy
       scipy
       astropy
       multiprocessing
       
Installation:
~~~~~~~~~~~~~
Not required.

    .. code:: bash
    
        $ git git@github.com:VasilyKokorev/ctf.git
        $ cd ctf
